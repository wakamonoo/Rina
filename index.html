Rina Welcomes You

A futuristic voice assistant extension for YouTube.

Download the extension and follow the steps to get started.

Download Extension
  1. Unzip the Rina.zip file to a folder.
  2. Go to chrome://extensions or edge://extensions.
  3. Enable Developer mode.
  4. Click Load unpacked.
  5. Select the folder containing manifest.json.
  6. The extension should now be active.
  7. Use voice commands on any YouTube tab.

Available Commands

Wake-up Call important

  • "Hey Rina-Chan", "Rina", "Hey", "Oy"

Playback

  • "play", "pause video", "stop video"

Volume

  • "volume up", "volume down"
  • "mute", "unmute"

Navigation

  • "skip", "forward", "back", "rewind"
  • "jump [time]" (e.g., "jump to 2:30")
  • "next", "previous"

Speed

  • "faster", "slower", "speed 1.5", "speed 2"

Fullscreen

  • "fullscreen", "exit fullscreen"

Search

  • "search [query]" (e.g., "search for trailers")
  • "play [query]" (e.g., "play lofi music")

* Commands are case-insensitive and designed to work on YouTube tabs.
